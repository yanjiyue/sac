## 筆記
### linked list
>prev指向前面節點|item數據|指向後面的節點
#### LinkedList 特點
 - 雙向鏈表實現
 - 元素時有序的，輸出順序與輸入順序一致
 - 允許元素為 null
 - 要找到某個結點，必須從頭開始遍曆。（查詢慢，增刪快）
 - 和 ArrayList 一樣，不是同步容器
